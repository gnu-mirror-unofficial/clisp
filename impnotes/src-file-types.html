33.1. File Types

#P"*.d"
The source files for unpreprocessed C code.
#P".c"
The C code after preprocessing.
#P".lisp"
The source files for Lisp code.
#P"*.fas"
Compiled lisp code (platform-independent).
#P".lib"
Lisp “header”, produced by COMPILE-FILE and used by REQUIRE
